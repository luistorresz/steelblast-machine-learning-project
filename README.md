
# Run in Google Colab

Use these steps to run the project notebooks in Google Colab.

## Steps

1. Open [Google Colab](https://colab.research.google.com/).
2. Click **Open notebook**.
3. Select the **GitHub** tab.
4. Enter this repository URL:

	```text
	https://github.com/luistorresz/steelblast-machine-learning-project.git
	```

5. Open the notebook named `run_in_colab.ipynb`.
6. In Colab, go to **Runtime > Change runtime type**.
7. Set **Hardware accelerator** to **T4 GPU**.
8. Click **Save**.
9. Run all notebook cells.

## Notes

- If the runtime disconnects, reconnect and run all cells again.
- If a package is missing, run the installation cell first and then continue.
